# Personal Portfolio

React + Vite personal website prepared for GitHub Pages.

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

This project includes a GitHub Actions workflow at `.github/workflows/deploy.yml`.
After the repository exists on GitHub, enable GitHub Pages with **GitHub Actions**
as the source. Every push to `main` will build and deploy the site.

For a user site, name the GitHub repository:

```text
<your-github-username>.github.io
```
